# NXLogis Station Data

## 소개
NXLogis.KR에 등록되어 있는 대한민국 국철 역 데이터입니다.   
한국철도공사 내부 시스템(XROIS)에 등록된 정거장 기준으로 하였으며, 모든 정보는 공공데이터로 분산 공개되어 있는 자료를 NXLogis에서 취합 및 가공하였습니다.   
일부 서울교통공사 관할 역이 포함되어 있지만 이는 한국철도공사 내부 시스템(XROIS)에 등록되어 있기 때문이며 대한민국의 모든 역(도시철도 포함)을 포함하는 취지 및 의미가 절대 아닙니다.   
이 데이터는 예고 없이 변경될 수 있습니다.   
데이터에 오류가 있으면 Issue 열어 주세요. 확인 후 처리하겠습니다.   

## 경고
 - NXLogis.KR과 이 데이터와 관련된 모든 철도운영사(예: 한국철도공사, 서울교통공사, 공항철도주식회사), 기타 유관 기관(예: 국토교통부, 국가철도공단) 는 이 데이터의 정확성, 유용성, 안정성, 확실성을 절대 보증하지 않습니다. 그리고 데이터의 사용으로 인해 발생한 손해에 대해 절대 책임지지 않습니다. 모든 책임은 사용자의 것입니다.   
 - 이 데이터는 철도운영사에서 직접 제공하는 것이 아니니 이 데이터를 가지고 철도운영사 및 유관 기관에 민원을 넣지 마시기를 바랍니다.
